# Password Redirect Website

Simple static site with a single password field. Typing a password and submitting navigates to `password.html` where `password` is the text entered (URL-encoded).

How to use:

1. Open `index.html` in your browser (`~/Documents/WebsitePassword/index.html`).
2. Enter a password (for example: `secret`) and click Go.
3. The browser will navigate to `secret.html` (if present) or show a 404 otherwise.

A sample `secret.html` is included.
